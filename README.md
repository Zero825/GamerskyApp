# GamerskyApp
    一个游民星空的第三方Android客户端
    应用截图在preview文件夹
    或者直接查看releases
### 功能特性
* 下拉刷新
* 上拉自动加载
* 阅读正文
* 查看评论
* 图片懒加载
### 引用
    查看大图引用[BigImageViewer](https://github.com/Piasy/BigImageViewer)
    下拉刷新引用[SmartRefreshLayout](https://github.com/scwang90/SmartRefreshLayout)
    图片懒加载引用[echo](https://github.com/toddmotto/echo)
